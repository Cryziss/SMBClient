# SMBClient

Swift SMB client library and iOS/macOS file browser applications.

## Supported SMB Messages

- [x] NEGOTIATE
- [x] SESSION_SETUP
- [x] LOGOFF
- [x] TREE_CONNECT
- [x] TREE_DISCONNECT
- [x] CREATE
- [x] CLOSE
- [ ] FLUSH
- [x] READ
- [x] WRITE
- [ ] LOCK
- [ ] ECHO
- [ ] CANCEL
- [x] IOCTL
- [x] QUERY_DIRECTORY
- [ ] CHANGE_NOTIFY
- [x] QUERY_INFO
- [x] SET_INFO
